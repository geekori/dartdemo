强类型编程语言和弱类型编程语言

强类型编程语言：
变量必须在编译时确定一个唯一的数据类型。如果数据类型
不一致，在编译时会抛出异常，进行类型检测。

弱类型编程语言：
变量的数据类型在运行时确定，而且变量的数据类型在运行时
是可以变化的。
例如，一个变量在初始化时是字符串类型，在运行时，可以将
一个整数赋给该变量，这时该变量就变成了整数类型。

而且，在弱类型编程语言中，是不需要为变量指定数据类型的。

强类型的特征：
1. 必须在代码中为变量指定数据类型。
（1）显式  int number = 20;
（2）隐式  var number = 20;  //推断数据类型为整数
  number = "abc" // error
2. 一个变量只能拥有一个数据类型，而且数据类型一旦指定，
无法改变
Dart、Swift、Kotlin、Go、Java、C++

弱类型编程语言
1. 不需要显式为变量指定数据类型
2. 变量的数据类型在程序的运行过程中可以改变，如从
字符串变成整型

JavaScript、Python、PHP

Dart
Swift
